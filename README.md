Secure Bookmarking Extension

Version: 1.0.0
Updated: November 26, 2025

Overview

Secure Bookmarking Extension is a browser extension that helps you organize and protect your bookmarks.

Key Features:

  Password-protected folders for sensitive bookmarks.

  Drag-and-drop sorting for intuitive bookmark management.

  Responsive UI with smooth animations and quick access.

  All data is stored locally for privacy — nothing is sent externally.

  The UI layout was inspired by an existing design but all code and functionality are original.

Installation

  1. Download or clone this repository.

  2. Load the extension in Chrome via chrome://extensions → "Load unpacked".

  3. Select the folder containing this extension.

Privacy

  No data is collected or sold.

  All bookmark data stays on your device.
